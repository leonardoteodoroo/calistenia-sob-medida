# 🏃‍♂️ CALISTENIA SOB MEDIDA - GUIA MAGIC MCP

Este projeto utiliza o **Magic MCP (21st.dev)**, um motor de IA de interface que funciona como um arquiteto sênior de frontend diretamente no seu chat. Este documento serve como bússola para você saber **o que pedir** e **como pedir**.

---

## 🛠️ Ferramentas Disponíveis (Tools)

O servidor MCP (`@21st-dev/magic`) disponibiliza as seguintes ferramentas para o seu agente (Antigravity):

| Ferramenta          | O que ela faz                                                         |
| :------------------ | :-------------------------------------------------------------------- |
| `create_ui`         | Cria componentes React novos do zero (Cards, Navbars, Gráficos, etc). |
| `search_components` | Busca inspirações e componentes prontos na biblioteca 21st.dev.       |
| `enhance_ui`        | (Beta) Melhora um componente que já existe no seu projeto.            |

---

## 💡 Como saber o que pedir? (O "Dicionário de Magia")

Se você não souber termos técnicos como "Framer Motion" ou "SVG", use esta tabela de tradução:

| Se você quer...        | Peça assim (Prompt)                                            | Resultado Esperado                                     |
| :--------------------- | :------------------------------------------------------------- | :----------------------------------------------------- |
| **Movimento Suave**    | "Peça com **micro-interações** ou **animações fluidas**"       | Botões que pulsação, cards que deslizam.               |
| **Ícones Bonitos**     | "Use **ícones modernos**"                                      | Ele usará a biblioteca Lucide (fogo, braços, relógio). |
| **Visual Prata/Vidro** | "Estilo **Glassmorphism**"                                     | Efeito de vidro transparente, muito premium.           |
| **Gráficos de Perfil** | "Mostre meus dados em um **gráfico de radar** ou **circular**" | Uma representação visual do seu nível de treino.       |
| **Cores que Combinam** | "Use uma **paleta de cores harmoniosa** (dark mode)"           | Tons de cinza, preto e acentos de cores neon.          |
| **Passo a Passo**      | "Crie um **Stepper** ou **Barra de Progresso**"                | Ideal para o seu Quiz de calistenia.                   |

## 📚 Bibliotecas que o Magic domina

O Magic já sabe usar as melhores ferramentas para o seu projeto Vite/React:

- **Tailwind CSS:** Estilos rápidos e responsivos.
- **Framer Motion:** A "mágica" das animações.
- **Lucide React:** O "alfabeto" de ícones.
- **shadcn/ui:** O "padrão ouro" de componentes de interface.
